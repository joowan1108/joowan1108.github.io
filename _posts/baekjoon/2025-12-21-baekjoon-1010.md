[문제](https://www.acmicpc.net/problem/1010)

# 접근 방법
mCn을 계산해야 한다.

# 정답 코드 1 (math library 사용하기)

math.comb는 combination의 총 개수를 계산해줌.

``` python
import sys
import math
input = sys.stdin.readline

T = int(input().strip())

for _ in range(T):
    n, m = map(int, input().strip().split())
    print(math.comb(m, n))

```

# 정답 코드 2 (조합 직접 계산하기)

조합을 공식대로 계산하기

``` python
import sys
input = sys.stdin.readline

"""
mCn을 계산해야함.
"""
def fact(n):
  if n==1 or n==0:
    return 1
  return n*fact(n-1)

t = int(input())

for _ in range(t):
  n,m = map(int, input().split())
  print(fact(m)//(fact(n) * fact(m-n)))

```
